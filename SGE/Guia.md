# Puesta en marcha rápida

## 1) Base de datos PostgreSQL
Cargar el esquema:

```bash
psql -U postgres -d parking -f backend/database/schema.sql
```

## 2) Backend
```bash
cd backend
npm install
npm run dev
```


## 3) Frontend
```bash
cd frontend
npm install
npm run dev
```
