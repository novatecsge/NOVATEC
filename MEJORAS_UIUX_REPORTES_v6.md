# Mejoras UI/UX y reportes · v6

## Cambios principales

- Rediseño visual global con tarjetas, sombras, radios, foco accesible, microinteracciones y layout responsivo.
- Sidebar y navbar modernizados respetando colores por rol.
- Tablas dentro de contenedores con scroll horizontal para evitar desbordamientos.
- Mapa con scroll vertical/horizontal, área desplazable, controles de zoom y contención visual.
- Dashboard de reportes con filtros por mes/año, KPIs superiores, gráficas de línea, barras y dona.
- Reporte PDF mantiene gráficas SVG y tablas profesionales.
- Mejoras de accesibilidad: focus-visible, reducción de movimiento para usuarios con preferencia reduce-motion.

## Validación

Se revisaron imports y renderizado del frontend con build de Vite.
